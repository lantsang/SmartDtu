# 开发环境配置

## 开发工具

* vscode

## `vscode` 插件列表

* Code Spell Checker (拼写检查插件，检查单词拼写错误，这个必须装，必须装，必须装，毋庸置疑！！！)
* Todo Tree (查看项目中TODO标记)
* Better Comments (更漂亮的注释显示)
* Guides (更好用的缩进参考线)
* koroFileHeader (生成文件头部注释，需要配置，默认生成全文件的)
* Python Indent (更好用的缩进参考线)
* Python Docstring Generator (Python代码注释插件)

> 已经在 `.vscode、extensions.json` 文件中将这些插件写入推荐配置，第一次打开时 `vscode` 会提示是否安装推荐插件，点击安装即可。 `Python` 项目不仅仅是安装了插件就行，还需要一些特定的配置，具体配置请转至笔记：[移远工程配置——vscode](https://gitlab.lantsang.cn/dongzhenguo/wiki/blob/master/%E7%A1%AC%E4%BB%B6/%E7%A7%BB%E8%BF%9C/%E5%B7%A5%E7%A8%8B%E9%85%8D%E7%BD%AE/vscode%E5%B7%A5%E7%A8%8B%E9%85%8D%E7%BD%AE.md)
